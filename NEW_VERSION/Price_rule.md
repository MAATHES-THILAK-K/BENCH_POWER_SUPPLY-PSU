# Final Price Rule

Final Price = ceil_up_to_5_or_10(max(((9 × m) + (0.75 × n)) × Layer × 1.15 + 15, 3.5 × n))

- m = time in hours
- n = weight in grams

Layer:
- 0.12 → 2.10
- 0.16 → 1.90
- 0.20 → 1.70
- 0.28 → 1.50

Double color:
- Multiply base cost by 2.2 before applying Layer

Rounding:
- Always round UP to nearest ₹5 or ₹10

Minimum:
- Final price must not be below 3.5 × grams
- Absolute minimum job charge: ₹30
