# 🔌 BENCH POWER SUPPLY – PSU (3D Model)

**PSU 3D Model designed in Fusion 360** for an **ATtiny85-based mini oscilloscope** and adjustable DC power system.

This project provides a **compact bench power supply enclosure** integrating voltage regulation, current measurement, user controls, and modular I/O — optimized for lab and embedded electronics use.

---

## 📦 Features & Supported Modules

- **ATtiny85 Mini Oscilloscope**
- **LM2596 Buck Converter**
  - Input: 4–40 V DC
  - Output: 1.3–37 V DC
  - Max Current: 2 A
- **Digital Voltage & Current Measurement Unit**
- **Potentiometer** for output voltage control
- **4 Switches** for oscilloscope & power control
- **Female Banana Plug Output**
- **USB Breakout Port**
- **DC Jack / Terminal Block Input**
